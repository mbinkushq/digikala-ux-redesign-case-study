# Digikala UX Redesign Case Study

A UX/UI redesign case study focused on improving product discovery, filtering, and decision-making experience.

## Project Overview

This project analyzes key UX challenges in Digikala and proposes redesign solutions to reduce cognitive load and improve user experience.

## Focus Areas

- Product Listing Experience
- Product Detail Page
- Filter & Search Refinement
